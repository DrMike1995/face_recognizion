// 01 — Funktioner

Allt du behöver
i ett program

Byggt med Python, OpenCV och dlib. Körs helt lokalt — ingen molntjänst, ingen nätverksanslutning krävs.

🔲

Realtidsdetektering

HOG-baserad ansiktsdetektor från dlib ritar färgade boxar runt varje ansikte i kameraflödet med ~30 fps.

👤

Automatiska ID:n

Varje ny unik person får automatiskt ett ID — Person 1, Person 2 osv. — baserat på 128-dimensionella face encodings.

👥

Multi-face tracking

Hanterar flera ansikten i bild samtidigt. Varje person får sin egen unika färg och spåras oberoende av de andra.

📸

Bildgalleri

Bilder sparas automatiskt per person. Klicka på en person i galleriet för att se alla sparade bilder i ett popup-fönster.

🔒

100% lokalt

All data stannar på din dator. Inga bilder eller encodings skickas till internet. Mappen captured_faces/ ignoreras av git.

⚙️

Konfigurerbart

Justera tolerans, skalfaktor, capture-intervall och mer direkt i toppen av main.py utan att ändra logiken.

// 02 — Teknisk översikt

Hur det fungerar

Pipeline från kameraframe till sparad bild i fyra steg.

01 —

Detektering

Varje frame skalas ner och skickas genom dlibs HOG-detektor som returnerar bounding box-koordinater för varje ansikte.

02 —

Encoding

face_recognition omvandlar varje ansikte till en 128-dimensionell vektor som unikt beskriver personens ansikte.

03 —

Matchning

Den nya encodingen jämförs mot alla kända med euklidiskt avstånd. Under 0.55 = samma person, annars ny buffras i 3 frames.

04 —

Lagring

Var 2:a sekund per person klipps ansiktet ut och sparas som JPEG i captured_faces/person_N/ lokalt på disken.

// 03 — Installation

Kom igång
på 3 minuter

Kräver Python 3.8–3.11 och en webbkamera. Windows-användare behöver Visual Studio Build Tools för dlib.

Mac / Linux
# Klona repot
git clone github.com/user/facetracker
cd facetracker

# Virtuell miljö
python -m venv venv
source venv/bin/activate

# Installera & kör
pip install -r requirements.txt
python src/main.py
Windows
# Klona repot
git clone github.com/user/facetracker
cd facetracker

# Virtuell miljö
python -m venv venv
venv\Scripts\activate

# Installera & kör
pip install -r requirements.txt
python src/main.py
🔒

Integritet & etik

Det här programmet är enbart avsett för skolbruk. All insamlad data lagras temporärt och lokalt — inga bilder, encodings eller persondata skickas till någon server. Radera mappen captured_faces/ för att ta bort all sparad data. Mappen ignoreras automatiskt av git via .gitignore.
